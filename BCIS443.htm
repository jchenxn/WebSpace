IS 443/543 DATABASE DESIGN AND IMPLEMENTATION

This course covers the database design process: conceptual design, logical design, and implementation. The course teaches students database design principles and techniques such as E.R.M. and normalization. Oracle 12c is used to demonstrate database implementation. Great emphasis is placed on using triggers to enforce business rules during implementation.

To access Oracle 12c Server, use a client program named Oracle SQL Developer in your classroom or CH 491 Lab and CH 450.
