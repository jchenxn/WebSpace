DESCRIPTION This course explores advanced database design topics. Current study topics include Web services and online analytical processing (OLAP), Online database application deployment, Data warehousing, Advanced database application modeling using unified modeling language (UML), Extensible Markup Language (XML), High availability system deployment among others.  Students taking this class should have completed BCIS443/543 Database Design and Implementation. A project involves in-depth studies in a selected topic area is required.
