MS IA 643 DATABASE SECURITY & AUDITING
This course is broken into three sections: database security fundamentals, article readings, secuirty and auditing implementation in Oracle10g. Students will be exposed to both discretionary and mandatory security models. The course will focus on user administration, discretionary access control, virtual database, and various auditing model implementation. Other topics covered in the class include SQL Injection, Statistical databases, Oracle encryption. Students will fully understand how to implement database security on modern business databases using practical scenarios and step-by-step examples.
